Web-технологии можно условно разделить на две пересекающиеся группы:клиентские и серверные

  • Клиентские:
    HTML
    стандартный язык разметки Web-страниц
    CSS
    средство описания и оформления Web-страниц
    JavaScript
    применяется в браузерах как язык сценариев для придания интерактивности Web-страниц
  • Серверные:
    ASP.NEY(C#)
    технология создания Web-приложений от компании Microsoft
    JSP(Java)
    технология создания Web-приложений от компании Oracle
    PHP
    язык программированния интенсивно применяемый для работы Web-приложений
